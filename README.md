# SportBookingApp

Instructions on connecting to db via Docker Desktop and MySQL Workbench:

1. Download Docker Desktop here: https://docs.docker.com/desktop/install/windows-install/
2. Download MySQL Workbench: https://dev.mysql.com/downloads/workbench/ (v8.0.36)
3. Move to designated directory
4. Git clone this repo
5. cd to "SportBookingApp"
6. docker-compose up
